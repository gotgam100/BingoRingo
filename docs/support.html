❓ 자주 묻는 질문 / FAQ

Q. 빙고 방에 어떻게 참여하나요?

방장이 공유한 초대 링크 또는 초대 코드를 통해 참여할 수 있습니다.

Q. How do I join a bingo room?

You can join by using the invite link or invite code shared by the room leader.

Q. 무료로 몇 개의 빙고 방을 만들 수 있나요?

무료 사용자는 1개의 빙고 방을 만들 수 있습니다. 프리미엄 구매 시 무제한으로 생성 가능합니다.

Q. How many bingo rooms can I create for free?

Free users can create 1 bingo room. Purchasing Premium allows unlimited room creation.

Q. 프리미엄을 구매했는데 적용이 안 돼요.

설정 화면에서 '구매 복원' 버튼을 눌러 주세요. 문제가 지속되면 이메일로 문의해 주세요.

Q. I purchased Premium but it's not activated.

Please tap the 'Restore Purchases' button in Settings. If the issue persists, contact us via email.
